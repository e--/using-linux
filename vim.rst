vim
====================

basic config
----------------
::

    set tabstop=4 "tab键的宽度
    set expandtab "用空格替换tab键
    set shiftwidth=4 "设置缩进的空格数为4
    set nu "显示行号 
    set cindent "设置使用 C/C++ 语言的自动缩进方式

--------------------

    Ctrl+v
